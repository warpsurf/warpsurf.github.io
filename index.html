Vision

The browser is the door to the internet‚Äîit should be open and accessible. Warpsurf provides a model-agnostic AI copilot that can make complex workflows easier while keeping your data local.

Warpsurfing

For browser automation to be useful, it needs to be fast‚Äîenabling "warpsurfing". We're not there yet, but this is the goal. With faster models and parallel execution, we can get there.

Community

Warpsurf has been built as an open source community tool open to contributions and critique. We want to grow the browser automation ecosystem by finding bugs, useful features, and new use cases.

Safety

Warpsurf is a research project under active development. Browser automation carries inherent risks: agents may make mistakes and the web can be dangerous. Monitor agents while they work, use capped API keys, and use warpsurf at your own risk.
